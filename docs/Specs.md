# EBNF Specification of language I (Extended and Improved)
```
Program
  : { SimpleDeclaration | RoutineDeclaration }

SimpleDeclaration
  : VariableDeclaration
  | TypeDeclaration

VariableDeclaration
  : var Identifier : Type [ is Expression ]
  | var Identifier is Expression

TypeDeclaration
  : type Identifier is Type
```
```
Type
  : PrimitiveType
  | UserType
  | Identifier

PrimitiveType
  : integer | real | boolean

UserType
  : ArrayType | RecordType

RecordType
  : record { VariableDeclaration } end

ArrayType
  : array [ [ Expression ] ] Type
  ```
```
Statement
  : Assignment
  | RoutineCall
  | WhileLoop
  | ForLoop
  | IfStatement
  | PrintStatement

Assignment
  : ModifiablePrimary := Expression

RoutineCall
  : Identifier [ ( Expression { , Expression } ) ]

WhileLoop
  : while Expression loop Body end

ForLoop
  : for Identifier in Range [ reverse ] loop Body end

Range
  : Expression [ .. Expression ]

IfStatement
  : if Expression then Body [ else Body ] end

PrintStatement
  : print Expression { , Expression }
```
```
RoutineDeclaration
  : RoutineHeader [ RoutineBody ]

RoutineHeader
  : routine Identifier ( [ Parameters ] ) [ : Type ]

RoutineBody
  : is Body end
  | => Expression

Parameters
  : ParameterDeclaration { , ParameterDeclaration }

ParameterDeclaration
  : Identifier : Type
```
```
Body
  : { SimpleDeclaration | Statement }
 ```
```
Expression
  : Relation { ( and | or | xor ) Relation }

Relation
  : Simple [ ( < | <= | > | >= | = | /= ) Simple ]

Simple
  : Factor { ( + | - ) Factor }

Factor
  : Summand { ( * | / | % ) Summand }

Summand
  : Primary | ( Expression )

Primary
  : [ Sign | not ] IntegerLiteral
  | [ Sign ] RealLiteral
  | true
  | false 
  | ModifiablePrimary
  | RoutineCall 

Sign
  : + | -

ModifiablePrimary
  : Identifier { . Identifier | [ Expression ] }
  
 ```
